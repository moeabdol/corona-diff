# Corona Virus Strain Diff
This app shows a plot of mutation between two COVID-19 strains.

## Instructions
1. `python3 -m venv .venv`
2. `pip install --upgrade pip`
3. `pip install -r requirements.txt`
4. `source .venv/bin/activate`
5. `python main.py`
